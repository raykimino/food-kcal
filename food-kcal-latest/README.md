# 食物热量记录 - Vue 3 重构版 🍎

这是原食物热量记录应用的 Vue 3 重构版本，使用组合式 API 和 TypeScript 实现。功能界面完全保持一致，但采用了现代的前端技术栈。

## 🚀 功能特性

与原版本完全相同的功能特性：
- **饮食日历** — 月历视图，有记录的日期显示绿点，点击切换查看
- **分餐段记录** — 早餐 / 午餐 / 晚餐 / 加餐零食四个餐段独立统计
- **三大营养素** — 可选填蛋白质、碳水化合物、脂肪，当日汇总一览
- **每日目标进度条** — 自定义热量目标，超标变红、接近变橙
- **收藏夹快选** — 收藏常吃食物，一键填充名称与营养信息
- **历史记忆自动补全** — 再次输入相同食物名称自动填入上次热量
- **复制记录** — 将任意日期的全部记录一键复制到当前日期
- **快速克隆** — 单条食物旁一键复制，快速添加重复项
- **批量删除** — 多选模式批量删除食物条目
- **热量趋势图表** — 近7天 / 本周 / 本月三种视角，含总计、日均、有记录天数统计
- **完成打卡** — 打卡按钮触发撒花庆祝动画 + 手机震动反馈
- **数据导出 / 导入** — JSON 格式备份与恢复，兼容旧版数据格式
- **深色模式** — 自动跟随系统主题切换
- **PWA 支持** — 可安装到手机主屏幕，支持离线使用

## 🛠️ 技术栈

- **Vue 3** - Composition API
- **TypeScript** - 类型安全
- **Pinia** - 状态管理
- **Chart.js** - 数据可视化
- **Lucide Vue** - 图标库
- **Canvas Confetti** - 撒花特效
- **Vite** - 构建工具

## 🔄 数据兼容性

### 数据格式
本版本与原版本使用相同的数据存储格式，存储在浏览器的 `localStorage` 中：

| 键名 | 内容 |
|------|------|
| `foodTrackerDataV2` | 所有饮食记录（按日期索引） |
| `foodTrackerGoal` | 每日热量目标（默认 2000 kcal） |
| `foodTrackerFavorites` | 收藏的常用食物列表 |

### 数据迁移
1. **导出原版本数据**：在原应用中点击"导出 JSON"
2. **导入到新版本**：在新应用中点击"导入 JSON"
3. **数据完全兼容**：所有数据字段和格式保持一致

## 🎨 技术特点

1. **响应式设计**：移动端优先，自适应各种屏幕尺寸
2. **组合式 API**：使用 Vue 3 Composition API，逻辑复用性好
3. **TypeScript**：完整的类型定义，开发体验更佳
4. **模块化组件**：每个功能模块独立组件，维护方便
5. **状态管理**：使用 Pinia 进行集中状态管理
6. **PWA 支持**：可安装到手机主屏幕，支持离线使用
7. **深色模式**：自动跟随系统主题切换

|------|---------------------|-------------|

## 🔧 开发说明

### 主要组件说明

1. **App.vue** - 主组件，管理主题和全局布局
2. **Calendar.vue** - 日历组件，展示月份和日期选择
3. **FoodForm.vue** - 表单组件，添加/编辑食物记录
4. **DailySummary.vue** - 每日摘要组件，展示统计信息
5. **MealDetails.vue** - 饮食明细组件，展示餐段记录
6. **ChartStats.vue** - 图表统计组件，展示趋势图表

### 状态管理

使用 Pinia 管理全局状态，包括：
- 食物数据
- 收藏夹
- 选中日期
- 图表视图
- 批量选择模式

### 数据持久化

所有数据存储在 `localStorage` 中，支持：
- 自动保存
- 数据导出/导入
- 向前兼容旧版本数据
